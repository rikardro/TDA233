from sklearn.metrics.pairwise import euclidean_distances
class K_Nearest_Neighbors():
    """
    Custom implementation of K-nearest-neighbors.

    Note: If two or more classes have the same amount of neighbors, the prediction
    will be random, i.e. tiebreaks are resolved randomly.
    """

    def __init__(self, n_neighbors):
        """
        Parameter(s):
            n_neighbors - Number of neighbors
        """
        self.n_neighbors = n_neighbors

    def fit(self, X_train, y_train):
        """
        No training required. Store data and labels.
        --------------------
        Input(s):
            X_train   - Data of size (n_samples, n_features)
            y_train   - True labels of size (n_samples,1)
        --------------------
        Output(s)
        """
        self.X_train = X_train
        self.y_train = y_train

    def predict(self, X_test):
        """
        Prediction of test data.
        --------------------
        Input(s):
            X_test   - Data of size (n_samples, n_features)
        --------------------
        Output(s)
            y_pred - Predicted labels of size (n_samples,)
            y_pred_prob - Probabilistic labels of size (n_samples,n_classes)
        """
        n_samples = X_test.shape[0]
        unique_classes = np.unique(self.y_train)
        n_classes = len(unique_classes)

        y_pred = np.zeros(shape=(n_samples,), dtype=np.int32)
        y_pred_prob = np.zeros(shape=(n_samples, n_classes), dtype=np.float32)

        # Computes distances between all points in X_test and X_train.
        dist = euclidean_distances(X_test,self.X_train)

        for i in range(n_samples):
            dist_i = dist[i]

            # Get labels of the k nearest neighbors
            nearest_indices = np.argsort(dist_i)[:self.n_neighbors]
            nearest_labels = self.y_train[nearest_indices]
            
            # Count occurrences of each class
            unique_labels, counts = np.unique(nearest_labels, return_counts=True)
            class_counts = np.zeros(n_classes, dtype=np.float32)
            class_counts[unique_labels] = counts
            
            # Compute probability distribution
            y_pred_prob[i, :] = class_counts / self.n_neighbors

            # Selects the prediction randomly, based on y_pred_prob[i].
            y_pred[i] = np.random.choice(unique_classes, p=y_pred_prob[i])

        return y_pred, y_pred_prob

2. Naive Bayes Classifier (7 pts)

The Bayes classifier refers to a classifier using the predicitive probability: $$ P(y_{n} = c | x_{n}, X, y) = \frac{p(x_{n}|y_{n} = c, X, y)P(y_{n} = c | X, y)}{p(x_{n}|X,y)}, \tag{1} $$ where $X,y$ is the training data, $x_{n}$ a new test instance and $c$ its corresponding class.

Assume that the class-conditional densities are Gaussian, that is $$ p(x_{n}|y_{n} = c, X, y) = \mathcal{N}(\mu_{c}, \Sigma_{c}). $$

Furthermore, assume a class size prior, i.e. the prior is set using the proportion of training points in class $c$: $$ P(y_{n} = c |X, y ) = \frac{N_c}{N}, \tag{2} $$ where $N_c$ is the number of training objects belonging to class $c$, and $N$ is the total number of training objects.

Exercise 2.1 (5 pts)

We consider a binary classification problem with real valued data, i.e. $x_n = [x_{n1}, x_{n2}]^T \in \mathbb{R}^2$ and $c=\{0,1\}$.

1. (2 pts)

In order to simplify the problem, we use the Naive Bayes assumption, $$ p(x_{n}|y_{n} = c, X, y) = \prod_{d=1}^2 p(x_{nd}|y_n = c, X, y) = \prod_{d=1}^2 \mathcal{N}(\mu_{cd}, \sigma_{cd}^2), $$ which assumes that the two attributes of $x_n$ are independent of one another, i.e. the covariance matrix $\Sigma_{c}$ is diagonal.

Using this assumption, write the expression for the naive Bayes classifier, that is, derive $$ P(y_{n} = c | x_{n} , X, y ). $$

HINT: Derive the maximum likelihood estimates for the parameters $\mu_{c}, \Sigma_{c}$. Then express eq. 1 in terms of those estimates.

2. (3 pts)

Derive the maximum likelihood estimate for $\mu_{c}, \Sigma_{c}$ when the covariance matrix is not diagonal, i.e, $\Sigma_{c}$ has 4 unknown scalars. This alleviates the "naive" assumption, since the feature components are no longer independent of one another.

HINT: When deriving the MLE for $\Sigma_{c}$ it might be a good idea to consider $\partial /\partial \Sigma_{c}^{-1}$. Also, feel free to use the following properties: $$ \frac{\partial\ x^T A x}{\partial A} = x x^T, \qquad \frac{\partial\ x^T A x}{\partial x} = 2Ax, \qquad \frac{\partial\ \log |A|}{\partial A} = A^{-T}, $$ for a symmetric matrix $A$ and a vector $x$ that does not depend on $A$.

1.

We want to compute:

$$ P(y_{n} = c | x_{n}, X, y) = \frac{p(x_{n}|y_{n} = c, X, y)P(y_{n} = c | X, y)}{p(x_{n}|X,y)} $$

From the assumptions, we have: $$ P(y_{n} = c |X, y ) = \frac{N_c}{N} $$ $$ p(x_{n}|y_{n} = c, X, y) = \prod_{d=1}^2 \mathcal{N}(\mu_{cd}, \sigma_{cd}^2) $$

where

$$ \mathcal{N}(\mu_{cd}, \sigma_{cd}^2) = \frac{1}{\sqrt{2\pi \sigma_{cd}^2}} \exp\left \{ -\frac{(x_{nd} - \mu_{cd})^2}{2\sigma_{cd}^2} \right\}. $$

Deriving the MLE's for $\mu_{c}, \Sigma_{c}$:

The likelihood function is given by:

$$ \mathcal{L}(\mu_c, \Sigma_c) = \prod_{n=1}^{N_c} \prod_{d=1}^2 \mathcal{N}(\mu_{cd}, \sigma_{cd}^2) = \prod_{n=1}^{N_c} \prod_{d=1}^2 \frac{1}{(2\pi \sigma^2_{cd})^{1/2}} \exp \left\{ -\frac{(x_{nd} - {\mu_{cd}})^2}{2 \sigma^2_{cd}} \right\}. $$

To find the maximum likelihood estimator $ \hat{\mu}$ and $\hat{\Sigma_{c}}$, we take the partial derivative of $\mathcal{L}(\mu, \Sigma_{c})$ w.r.t either variable and set it to zero.

We can simplify the results by differentiating the log-likelihood $\log \mathcal{L}(\mu, \Sigma_{c})$, which works since maximizing $\mathcal{L}(\mu, \Sigma_{c})$ will yield the same as maximizing $\log \mathcal{L}(\mu, \Sigma_{c})$ since the logarithm is a monotonic function.

$$ \log \mathcal{L}(\mu_c, \Sigma_c) = \sum_{n=1}^{N_c} \sum_{d=1}^2 \log \mathcal{N}(\mu_{cd}, \sigma_{cd}^2) = \sum_{n=1}^{N_c} \sum_{d=1}^2 \left ( -\frac{1}{2}\log(2\pi \sigma^2_{cd}) - \frac{(x_{nd} - {\mu_{cd}})^2}{2\sigma^2_{cd}} \right ) $$

$$ \textbf{Differentiate w.r.t. } \mu_{cd}: \quad \frac{\partial}{\partial \mu_{cd}} \log \mathcal{L}(\mu_c, \Sigma_c) = \frac{\partial}{\partial \mu_{cd}} \sum_{n=1}^{N_c} \sum_{d=1}^2 \left( -\frac{(x_{nd} - {\mu_{cd}})^2}{2\sigma^2_{cd}} \right) = \sum_{n=1}^{N_c} \frac{x_{nd} - \mu_{cd}}{\sigma^2_{cd}} $$ $$ \textbf{set derivate to 0 and solve for } \mu_{cd} :\quad \sum_{n=1}^{N_c} \frac{x_{nd} - \mu_{cd}}{\sigma^2_{cd}} = 0 \implies \mu_{cd} = \frac{1}{N_c} \sum_{n=1}^{N_c} x_{nd} \implies \hat{\mu}_{c} = \frac{1}{N_c} \sum_{n=1}^{N_c} x_n $$ $$ \textbf{Differentiate w.r.t. } \sigma^2_{cd}: \quad \frac{\partial}{\partial \sigma^2_{cd}} \log \mathcal{L}(\mu_c, \Sigma_c) = \sum_{n=1}^{N_c} \sum_{d=1}^2 \frac{\partial}{\partial \sigma^2_{cd}} \left[ -\frac{1}{2} \log(2\pi\sigma^2_{cd}) \right] - \frac{\partial}{\partial \sigma^2_{cd}} \left[ -\frac{(x_{nd} - {\mu_{cd}})^2}{2\sigma^2_{cd}} \right] = $$

$$ = \sum_{n=1}^{N_c} \sum_{d=1}^2 \left ( -\frac{1}{2\sigma^2_{cd}} + \frac{(x_{nd} - {\mu_{cd}})^2}{2\sigma^4_{cd}} \right ) = \sum_{n=1}^{N_c} \left ( -\frac{1}{2\sigma^2_{cd}} + \frac{(x_{nd} - \mu_{cd})^2}{\sigma^4_{cd}} \right ) $$

$$ \textbf{set derivate to 0 and solve for } \sigma^2_{cd} :\quad $$

$$ \sum_{n=1}^{N_c} \left ( -\frac{1}{2\sigma^2_{cd}} + \frac{(x_{nd} - \mu_{cd})^2}{\sigma^4_{cd}} \right ) = 0 \implies \sum_{n=1}^{N_c} \left ( -2\sigma^2_{cd} + (x_{nd} - \mu_{cd})^2 \right ) = 0 \implies $$

$$ \implies \sigma^2_{cd} = \frac{1}{N_c} \sum_{n=1}^{N_c} (x_{nd} - \mu_{cd})^2 \implies \hat{\Sigma}_c = \begin{bmatrix} \hat{\sigma}_{c1}^2 & 0 \\ 0 & \hat{\sigma}_{c2}^2 \end{bmatrix} $$

We need to expand $p(x_{n}|X,y)$. From the law of total probability, we can rewrite the expression:

$$ p(x_{n}|X,y) = \sum_{c \in \{0,1\}}{p(x_{n}|y_n = c, X,y)P(y_n = c | X, y)} = \sum_{c \in \{0,1\}} \left( \prod_{d=1}^2 \mathcal{N}(\mu_{cd}, \Sigma_{cd})\right) \frac{N_c}{N}. $$

From the above simplifications and assumptions, we get the following:

$$ P(y_{n} = c | x_{n}, X, y) = \frac{\prod_{d=1}^2 \mathcal{N}(\hat{\mu}_{cd}, \hat{\sigma}_{cd}^2) \frac{N_c}{N}}{\sum_{\tilde{c} \in \{0,1\}} \left( \prod_{d=1}^2 \mathcal{N}(\hat{\mu}_{\tilde{c}d}, \hat{\sigma}^2_{\tilde{c}d})\right) \frac{N_{\tilde{c}}}{N}} $$

2.

From the assumptions, we have: $$ p(x_{n}|y_{n} = c, X, y) = \mathcal{N}(\mu_{c}, \Sigma_{c}) $$

I.e., the likelihood that $y_n = c \text{ for } x_n$. From there, we derive the likelihood function:

$$ \mathcal{L} (\mu_{c}, \Sigma_{c}) = \prod_{i=1}^{N_c}{\mathcal{N}(\mu_{c}, \Sigma_{c})} $$

where

$$ \mathcal{N}(\mu, \Sigma) = \frac{1}{(2\pi)^{p/2} |{\Sigma}|^{1/2}} \exp \left\{ -\frac{1}{2} (\mathbf{x} - {\mu})^\top {\Sigma}^{-1} (\mathbf{x} - {\mu}) \right\} $$

To find the MLE's, we maximize the log-likelihood function:

$$ \log \mathcal{L} (\mu_{c}, \Sigma_{c}) = \sum_{i=1}^{N_c} \log \left[ \frac{1}{(2\pi)^{p/2} |{\Sigma_c}|^{1/2}} \exp \left\{ -\frac{1}{2} (\mathbf{x_i} - {\mu_c})^\top {\Sigma_c}^{-1} (\mathbf{x_i} - {\mu_c}) \right\} \right] $$

$$ = - \frac{N_c p}{2} \log(2 \pi) - \frac{N_c}{2} \log |\Sigma_c| - \frac{1}{2} \sum_{i=1}^{N_c} (\mathbf{x_i} - {\mu_c})^\top {\Sigma_c}^{-1} (\mathbf{x_i} - {\mu_c}). $$

MLE for $\mu_c$:

$$ \frac{\partial}{\partial \mu_c} \log \mathcal{L} (\mu_{c}, \Sigma_{c}) = \frac{\partial}{\partial \mu_c} - \frac{1}{2} \sum_{i=1}^{N_c} (\mathbf{x_i} - {\mu_c})^\top {\Sigma_c}^{-1} (\mathbf{x_i} - {\mu_c}) $$

Using the derivation rule $\frac{\partial\ x^T A x}{\partial x} = 2Ax$:

$$ \frac{\partial}{\partial \mu_c} - \frac{1}{2} \sum_{i=1}^{N_c} (\mathbf{x_i} - {\mu_c})^\top {\Sigma_c}^{-1} (\mathbf{x_i} - {\mu_c}) = \sum_{i=1}^{N_c}{\Sigma_c}^{-1} (\mathbf{x_i} - {\mu_c}) $$

Setting to zero and solving for $\mu_c$:

$$ \sum_{i=1}^{N_c}{\Sigma_c}^{-1} (\mathbf{x_i} - {\mu_c}) = 0 \implies \mu_c = \frac{1}{N_c} \sum_{i=1}^{N_c} \mathbf{x_i} $$

MLE for $\Sigma_c$:

$$ \frac{\partial}{\partial \Sigma_c^{-1}} \log \mathcal{L} (\mu_{c}, \Sigma_{c}) = \frac{\partial}{\partial \Sigma_c^{-1}} \left( - \frac{N_c}{2} \log |\Sigma_c| - \frac{1}{2} \sum_{i=1}^{N_c} (\mathbf{x_i} - {\mu_c})^\top {\Sigma_c}^{-1} (\mathbf{x_i} - {\mu_c}) \right) $$

We use the derivation rules $\frac{\partial\ \log |A|}{\partial A} = A^{-T}$ and $\frac{\partial\ x^T A x}{\partial A} = x x^T$. Since $\Sigma_c$ is symmetric, we have that $\frac{\partial}{\partial \Sigma_c^{-1}} \log |\Sigma_c| = \Sigma_c$.

$$ \frac{\partial}{\partial \Sigma_c^{-1}} \left( - \frac{N_c}{2} \log |\Sigma_c| - \frac{1}{2} \sum_{i=1}^{N_c} (\mathbf{x_i} - {\mu_c})^\top {\Sigma_c}^{-1} (\mathbf{x_i} - {\mu_c}) \right) = \frac{N_c}{2} \Sigma_c - \frac{1}{2} \sum_{i=1}^{N_c} (\mathbf{x_i} - {\mu_c})(\mathbf{x_i} - {\mu_c})^\top $$

Setting to zero and solving for $\Sigma_c$:

$$ \frac{N_c}{2} \Sigma_c - \frac{1}{2} \sum_{i=1}^{N_c} (\mathbf{x_i} - {\mu_c})(\mathbf{x_i} - {\mu_c})^\top = 0 \implies \Sigma_c = \frac{1}{N_c} \sum_{i=1}^{N_c} (\mathbf{x_i} - {\mu_c})(\mathbf{x_i} - {\mu_c})^\top. $$

class GaussianNaiveBayes():

        """

        Input(s):
            X_class - Data points corresponding to a single class
        --------------------
        Output(s):
            mean_MLE - MLE of mean
            std_MLE  - MLE of scale

        std_MLE = np.std(X_class, axis=0)

                    for d in range(X_test.shape[1]):
                        x = x_n[d]
                        mean = self.mean[c_prime][d]
                        std = self.std[c_prime][d]
                        densities_2.append(self.gaussian_density(x, mean, std))
                    likelihood_2 = np.prod(densities_2)
                    # P(y_n = c | X, y) - prior
                    prior_2 = self.prior[c_prime]
                    denominator.append(likelihood_2 * prior_2)
                denominator = np.sum(denominator)
                prob[c] = (likelihood * prior / denominator)

            y_pred_prob[n,:] = prob


        y_pred = np.argmax(y_pred_prob, axis=-1)
        return y_pred, y_pred_prob

3. SVM (5 pts)

Excercise 3.1 (3 pts)

The SVM classifier is most often phrased as the constrained optimization problem: $$ \begin{aligned} \underset{w,b}{\arg\min}\ & \frac{1}{2} w^T w \\ s.t. \quad & t_n(w^T x_n + b) \geq 1, \forall n, \end{aligned} $$ where $t_n \in \{-1,1\}$ are now the associated labels ($y_n = 0 \iff t_n = -1$; $y_n = 1 \iff t_n = 1$).

By moving the constraints into the objective function, we may formulate the problem using the hinge loss function: $$ J = \frac{1}{2} w^T w + C\frac{1}{N} \sum_{n=1}^N \max(0, 1 - t_n(w^T x_n + b)), $$ with $C$ as a regularization parameter.

Even though the hinge loss function is not differentiable, it is still a convex function and has subgradients with respect to both $w$ and $b$. Thus, it can still be used for optimization. Derive the following subgradients: $$ \frac{\partial J}{\partial w}, \quad \frac{\partial J}{\partial b}. $$

$$ \frac{\partial J}{\partial w} = w + C\frac{1}{N} \sum_{n=1}^N \frac{\partial}{\partial w} \max(0, 1 - t_n(w^T x_n + b)) $$

$$ \text{ where } \frac{\partial}{\partial w} \max(0, 1 - t_n(w^T x_n + b)) = \begin{cases} 0, & \text{if } t_n(w^\top x_n + b) > 1 \\ -t_n x_n, & \text{otherwise} \end{cases} $$

$$ \frac{\partial J}{\partial b} = C \frac{1}{N}\sum_{n=1}^N \frac{\partial}{\partial b} \max(0, 1 - t_n(w^\top x_n + b)) $$

$$ \text{ where } \frac{\partial}{\partial b} \max(0, 1 - t_n(w^\top x_n + b)) = \begin{cases} 0, & \text{if } t_n(w^\top x_n + b) > 1 \\ -t_n, & \text{otherwise} \end{cases} $$

import numpy as np

class SVM():
    """
    Custom implementation of linear SVM.
    """
    def __init__(self, learning_rate, n_iterations, C = 1, print_cost=False):
        """
        Parameter(s):
            learning_rate - Learning rate
            n_iterations  - Number of iterations
            C             - Regularization parameter
        """
        self.learning_rate = learning_rate
        self.n_iterations = n_iterations
        self.C = C
        self.print_cost = print_cost
        self.parameters = {}

    def initialize_parameters(self, n_features):
        """
        Initialize model parameters with zeros:
            w.shape = (n_features, 1)
            b.shape = (1,)
        --------------------
        Input(s):
            n_features - Number of features
        --------------------
        Output(s):
        """

        w = np.zeros(shape = (n_features,1))
        b = 0

        self.parameters["w"] = w
        self.parameters["b"] = b


    def grad_cost(self, X_train, t, w, b):
        """
        Computes the cost function (hinge loss) and
        partial derivatives of cost w.r.t the model parameters.
        --------------------
        Input(s):
            X_train - Data of size(n_samples, n_features)
            t       - Labels (-1 or 1) of size (n_samples,)
            w,b     - Model parameters
        --------------------
        Output(s):
            cost  - hinge loss
            grads - Gradients of loss function w.r.t model parameters (dw,db).
        """
        n_samples = X_train.shape[0]
        C = self.C
        w = w.ravel()  # (n_features,)

        sum = np.sum(np.maximum(0, 1 - t * (np.dot(X_train, w) + b)))
        J = 1/2 * np.dot(w.T, w) + C/n_samples * sum

        cost = J
        sum_dw = np.sum(
            np.array(
                [np.zeros_like(X_train[n]) if t[n] * np.dot(w.T, X_train[n]) + b > 1 else -t[n] * X_train[n] for n in range(n_samples)]), axis=0)
        dw = w + C/n_samples * sum_dw

        sum_db = np.sum(
            np.array(
                [0 if t[n] * np.dot(w.T, X_train[n]) + b > 1 else -t[n] for n in range(n_samples)]), axis=0)
        db = C/n_samples * sum_db

        grads = {"dw": dw.reshape(-1, 1),
                 "db": db}

        return cost, grads

    def fit(self, X_train, y_train):
        """
        Optimize model parameters by running a gradient descent algorithm.
        --------------------
        Input(s):
            X_train - Data of size (n_samples, n_features)
            y_train - True labels of size (n_samples,1)
        --------------------
        Output(s)
        """
        n_features = X_train.shape[1]

        # Change the binary targets from (0,1) to (-1,1)
        t = np.where(y_train < 0.5, -1, 1)

        # Initialize parameters
        self.params = self.initialize_parameters(n_features)

        # Gradient descent
        w = self.parameters["w"]
        b = self.parameters["b"]
        for i in range(1,self.n_iterations+1):
            cost, grads = self.grad_cost(X_train, t, w, b)
            dw = grads["dw"]
            db = grads["db"]

            w -= self.learning_rate * dw
            b -= self.learning_rate * db

            if self.print_cost and i % 100 == 0:
                print("Cost after iteration %i: %f" %(i, cost))

        self.parameters = {"w": w,
                           "b": b}

        return self

    def predict(self, X_test):
        """
        Binary prediction of test data.
        --------------------
        Input(s):
            X   - Data of size (n_samples, n_features)
        --------------------
        Output(s)
            y_pred - Predicted labels (0/1) of size (n_samples,1)
        """

        w = self.parameters["w"]
        b = self.parameters["b"]
        linear_output = np.dot(X_test, w) + b

        n_samples = X_test.shape[0]
        y_pred = np.zeros((n_samples,1))

        for i in range(n_samples):
            y_pred[i] = np.sign(linear_output[i])

        # Converts the binary targets from (-1,1) to (0,1)
        y_pred = (y_pred > 0).astype(np.int32)

        return y_pred.squeeze()

The last classifier that we will look at is Logistic Regression. Here you will finish a custom implementation of the Logistic Regression classifier using the following information:

  • The activation is computed as $$ A = \sigma(X w + b) = [a_1, \ldots, a_N]^T, $$ where $\sigma(z)$ is the sigmoid-function.

  • Derivatives w.r.t the model parameters: $$ \frac{\partial J}{\partial w} = \frac{1}{N}X^T(A-Y), \qquad \frac{\partial J}{\partial b} = \frac{1}{N} \sum_{n=1}^N (a_n-y_n). $$

class LogisticRegression():
    """
    Custom implementation of (binary) Logistic Regression.
    """
    def __init__(self, learning_rate, n_iterations, print_cost=False):
        """
        Parameter(s):
            learning_rate - Learning rate
            n_iterations  - Number of iterations
        """
        self.learning_rate = learning_rate
        self.n_iterations = n_iterations
        self.print_cost = print_cost
        self.parameters = {}

    def initialize_parameters(self, n_features):
        """
        Initialize model parameters with zeros:
            w.shape = (n_features,)
            b.shape = (1,)
        --------------------
        Input(s):
            n_features - Number of features
        --------------------
        Output(s):
        """

        w = np.zeros(shape = (n_features,1))
        b = 0

        self.parameters['w'] = w
        self.parameters['b'] = b

    def sigmoid(self, z):
        """
        Sigmoid function
        --------------------
        Input(s):
            z -
        --------------------
        Output(s):
            s - sigmoid(z)
        """
        s = 1 / (1 + np.exp(-z))
        return s

    def grad_cost(self, X_train, y_train, w, b):
        """
        Computes the cost function (negative log-likelihood) and
        partial derivatives of cost w.r.t the model parameters.
        --------------------
        Input(s):
            X_train - Data of size(n_samples, n_features)
            y_train - True labels of size (n_samples,1)
            w,b     - Model parameters
        --------------------
        Output(s):
            cost  - Negative log-likelihood cost
            grads - Gradients of loss function w.r.t model parameters (dw,db).
        """
        n_samples = X_train.shape[0]

        y_train = y_train.reshape(-1,1)

        A = np.zeros((n_samples,1))
        for i in range(n_samples):
            a = self.sigmoid(np.dot(X_train[i], w) + b)
            A[i] = a

        sum = 0
        for n in range(n_samples):
            sum += y_train[n] * np.log(A[n]) + (1 - y_train[n]) * np.log(1 - A[n])
        cost = - sum / n_samples

        dw = np.dot((X_train.T), (A - y_train)) / n_samples

        sum_db = 0
        for n in range(n_samples):
            sum_db += A[n] - y_train[n]
        db = sum_db / n_samples

        grads = {"dw": dw,
                 "db": db}

        return cost, grads

    def fit(self, X_train, y_train):
        """
        Optimize model parameters by running a gradient descent algorithm.
        --------------------
        Input(s):
            X_train - Data of size (n_samples, n_features)
            y_train - True labels of size (n_samples,1)
        --------------------
        Output(s)
        """
        n_features = X_train.shape[1]

        # Initialize parameters
        self.params = self.initialize_parameters(n_features)

        # Gradient descent
        w = self.parameters["w"]
        b = self.parameters["b"]
        for i in range(1,self.n_iterations+1):
            cost, grads = self.grad_cost(X_train, y_train, w, b)
            dw = grads["dw"]
            db = grads["db"]

            w -= self.learning_rate * dw
            b -= self.learning_rate * db

            if self.print_cost and i % 100 == 0:
                print("Cost after iteration %i: %f" %(i, cost))

        self.parameters = {"w": w,
                           "b": b}

        return self

    def predict(self, X_test):
        """
        Binary prediction of test data.
        --------------------
        Input(s):
            X   - Data of size (n_samples, n_features)
        --------------------
        Output(s)
            y_pred - Predicted labels (0/1) of size (n_samples,)
        """

        w = self.parameters["w"]
        b = self.parameters["b"]

        n_samples = X_test.shape[0]
        y_pred = np.zeros((n_samples,1))

        A = np.zeros((n_samples,1))
        for i in range(n_samples):
            A[i] = self.sigmoid(np.dot(X_test[i], w) + b)
            y_pred[i] = 0 if A[i] <= 0.5 else 1

        return y_pred.squeeze(), A
